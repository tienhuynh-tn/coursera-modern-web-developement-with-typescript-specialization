# Modern Web Development with TypeScript Specialization

This repository contains coursework and practice projects for Coursera's Modern Web Development with TypeScript Specialization by Edureka.

The specialization focuses on building scalable, type-safe web applications with TypeScript across the full stack. It starts with TypeScript fundamentals, moves into object-oriented programming, then applies TypeScript in React frontend applications and Node.js/MongoDB backend systems.

## Course Structure

The specialization is organized as a 4-course series:

1. **Getting Started with TypeScript**
   - TypeScript setup and compiler configuration
   - Primitive and advanced types
   - Objects, enums, unions, functions, generics, and reusable utilities

2. **Advanced TypeScript with OOP**
   - Classes, interfaces, inheritance, and access modifiers
   - Encapsulation, abstraction, polymorphism, and reusable application design
   - Advanced type features such as utility types, conditional types, mapped types, and type guards

3. **Frontend Web Development with TypeScript**
   - React projects with TypeScript
   - Typed components, props, state, hooks, forms, and event handling
   - UI libraries and production-ready frontend patterns

4. **Backend Web Development with TypeScript**
   - Node.js and Express with TypeScript
   - Typed APIs, middleware, error handling, authentication, and RBAC
   - MongoDB/Mongoose integration and backend testing with Jest

## Current Project

### `first-typescript-project`

The first project is a basic TypeScript setup with:

- TypeScript compiler configuration in `tsconfig.json`
- Source files under `src/`
- Build output configured for `dist/`
- NPM scripts for building, running, and watching TypeScript files

Run it from the project folder:

```bash
cd first-typescript-project
npm install
npm run build
npm start
```

For development mode:

```bash
npm run dev
```

## Learning Goals

By working through this repository, the goal is to practice:

- Writing safer JavaScript with TypeScript's type system
- Configuring TypeScript projects with strict compiler settings
- Designing maintainable code with OOP principles
- Building typed React applications
- Developing typed backend APIs with Node.js, Express, and MongoDB
- Applying testing, validation, and authentication patterns in full-stack projects
